========================

REQUIREMENTS:
- .NET 6
- SQL Server
- Visual Studio 2022

========================

DATABASE SETUP:
1. Open SQL Server Management Studio
2. Create database: BookStoreDB
3. Run file: BookStore.sql

4. Update connection string in appsettings.json:
"DefaultConnection": "Server=YOUR_SERVER;Database=BookStoreDB;Trusted_Connection=True;"

========================

HOW TO RUN:
1. Open project in Visual Studio
2. Restore NuGet packages
3. Build solution: Ctrl + Shift + B
4. Press Ctrl + F5 to run


GITHUB:
https://github.com/JesuoaHieu/WebForm_allEx.git

========================

NOTES:
- If error occurs, check database connection
